# Dovahkiin-class---DnD-5e
This code (as JavaScript file) is made for the MPMB character sheet for DnD 5e. A custom class that tries to translate the Dragonborn from TES5: Skyrim to DnD. Includes 3 subclasses: Guardian of the College (Caster); Guardian of the Companions (Martial); Guardian of the Nightingale (Stealth Archer).
