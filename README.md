# Dovahkiin class - DnD 5e
This JavaScript file is made for the MPMB character sheet for DnD 5e. A custom class that tries to translate the Dragonborn from TES5: Skyrim to DnD. Includes 3 subclasses: Guardian of the College (Caster); Guardian of the Companions (Martial); Guardian of the Nightingale (Stealth Archer). 

On this Google Drive folder you'll find a PDF doc explaining the (sub)class and its abileties:
